    Variables and Data Types

    JavaScript has several data types, including numbers, strings, booleans, objects, and more.

    Example: Declaring Variables

    
const message = "Hello, world!";
let count = 10;
var isActive = true; // Avoid using var in modern JS
console.log(message, count, isActive);

    Working with Functions

    Functions are blocks of code designed to perform a particular task.

    Function Declaration vs. Expression

    • Function Declaration: Defined using the function keyword.
    • Function Expression: Defined as part of an expression, often assigned to a variable.
    • Arrow Functions: A concise way to write functions, introduced in ES6.

    Example: Arrow Function

    
const greet = (name) => {
    return `Hello, ${name}!`;
};
console.log(greet("Alice"));

const add = (a, b) => a + b;
console.log(add(5, 3));

    DOM Manipulation

    The Document Object Model (DOM) is a programming interface for web documents.

    Selecting Elements

    
const myDiv = document.getElementById('myId');
const paragraphs = document.querySelectorAll('p');
const firstHeading = document.querySelector('h1');

    Modifying Content

    You can change the text or HTML content of elements.

    
myDiv.textContent = "New text content";
firstHeading.innerHTML = "Updated Heading";

    React Basics

    React is a JavaScript library for building user interfaces.

    Components

    React applications are built from components, which are independent and reusable pieces of UI.

    Here's a simple functional component:

    
import React from 'react';

function Welcome(props) {
  return <h1>Hello, {props.name}</h1>;
}

export default Welcome;

    JSX

    JSX is a syntax extension for JavaScript. It looks like HTML and allows you to write React elements.

    JSX expressions are compiled to regular JavaScript function calls. 

    
const element = <h1>Hello, world!</h1>;
// This is equivalent to:
// const element = React.createElement('h1', null, 'Hello, world!');
